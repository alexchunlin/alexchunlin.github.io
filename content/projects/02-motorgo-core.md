# MotorGo Core

**The board behind the Robot Etch A Sketch**

<!-- status: Shipped · Demoed at Open Sauce -->
<!-- tags: Motor Control, Etch A Sketch, Open Sauce -->

The MotorGo Core evolved from the Mini into a more capable motor control platform — and proved itself by powering a robot Etch A Sketch that we brought to Open Sauce.

## What It Does

A versatile motor controller board designed for projects that need precise, repeatable motion control. The Core gives you the control loop fundamentals in a package that's ready to build on.

## Highlights

- **Robot Etch A Sketch** — Demonstrated at Open Sauce, drawing precise patterns with dual motor coordination
- Built for rapid prototyping of motor-driven projects
- Stepping stone from the Mini's single-motor design toward multi-axis control
